Acceso en Red Local

Para compartir este dashboard con otros dispositivos en tu red, necesitas servirlo desde un servidor web. Opciones rápidas:

🐍 Python (más fácil):
python -m http.server 8080

Luego accede desde otros dispositivos: http://[TU-IP]:8080/vylex.html

💚 Node.js (npx):
npx serve -l 8080
